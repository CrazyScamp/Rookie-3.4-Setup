## 🛡 REAL TIME PROTECTION (Windows Defender Exclusion)

1. Open your **C: Drive**
2. Create a folder named: `RSL`
3. Inside `RSL`, create another folder named: `Rookie`
4. On your PC, search for: **Virus & Threat Protection**
5. Under **Virus & Threat Protection Settings**, click **Manage Settings**
6. Scroll down to **Exclusions**
7. Click **Add or Remove Exclusions**
8. Click **Yes** if prompted
9. Click **+ Add an Exclusion** → select **Folder**
10. Navigate to `C:\`
11. Select the `RSL` folder you created
12. Click **Select Folder**

## 📦 INSTALLING ROOKIE

1. Move the downloaded .exe file into ```C:\RSL\Rookie```

2. Move the downloaded .json file into ```C:\RSL\Rookie```

3. Run: ```androidsideloader.exe```

4. Select ```PROVIDE PUBLIC CONFIG```

5. Select ```FROM FILE```

6. Select the ```public.json```

(You should now see the JSON config details)

7. Click ```APPLY```


Rookie should now connect to the updated servers using the modified `public.json`.
